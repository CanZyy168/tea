បច្ចេកវិទ្យាព័ត៌មាន ( IT ) គឺជាសំណុំនៃផ្នែកដែលពាក់ព័ន្ធដែលរួមបញ្ចូលប្រព័ន្ធកុំព្យូទ័រ កម្មវិធី ភាសាសរសេរកម្មវិធី និងទិន្នន័យ និងដំណើរការព័ត៌មាន និងការរក្សាទុក។ [1] IT ជាផ្នែកនៃ បច្ចេកវិទ្យាព័ត៌មាន និងទំនាក់ទំនង (ICT)។ [2] ប្រព័ន្ធ បច្ចេកវិទ្យាព័ត៌មាន ( ប្រព័ន្ធ IT ) ជាទូទៅគឺជា ប្រព័ន្ធព័ត៌មាន ប្រព័ន្ធ ទំនាក់ទំនង ឬនិយាយឱ្យកាន់តែច្បាស់ជាងនេះទៅទៀត ប្រព័ន្ធកុំព្យូទ័រ — រួមទាំង ផ្នែករឹង សូហ្វវែរ និង ឧបករណ៍ គ្រឿងកុំព្យូទ័រ ទាំងអស់ — ដំណើរការដោយក្រុមអ្នកប្រើប្រាស់ព័ត៌មានវិទ្យាដែលមានកម្រិត ។ ហើយ គម្រោង IT ជាធម្មតាសំដៅទៅលើការចាត់តាំង និងការអនុវត្តប្រព័ន្ធ IT ។
